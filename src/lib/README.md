# Lib Directory

Library utilities and third-party integrations.

## Files

- **utils.ts**: cn() utility for className merging (shadcn/ui)
- **supabase.ts**: Supabase client configuration
- **validation.ts**: Form validation helpers
- **date.ts**: Date formatting and manipulation (date-fns wrappers)
- **format.ts**: Data formatting utilities

## Guidelines

1. Pure functions only
2. Well-tested utilities
3. No side effects
4. Clear documentation
5. Reusable across the app
